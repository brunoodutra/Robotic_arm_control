Inicialmente faça o upload do código do arduino que está na pasta
As portas analógicas utilizadas são a A0 até A6  e as portas PWM pros atuadores são as portas 8 até 13. 
Obs: cada as portas PWM e analógicas correspondem entre si para as relações de entrada e saída de cada modelo (servo/posição angular).


O arquivo principal é o "tutorial_ident_multivar.m".
Caso prefira, o tutorial pode ser seguido passo a passo a partir do arquivo "tutorial_ident_multivar.htlm" que está na pasta "tutorial_step_by_step" 
ou pelo aquivo pdf. 

No final da aquisição vão ser identificação 3 modelos. O melhor será utilizado para a sintonia do controlador
